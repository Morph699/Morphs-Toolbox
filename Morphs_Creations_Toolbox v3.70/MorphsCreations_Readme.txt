🛠️ ESSENTIAL POST-INSTALL RUNTIME INSTRUCTIONS
================================================================================

1. WINDOWS DEFENDER UTILITY WIDGET
   - To prevent long installation scan queues during custom application extraction, 
     use the bundled Defender Control tool to toggle real-time security scanning.
   - Run the desktop guide utility for a complete visual walkthrough: 
     https://youtu.be/Ht7o569mFeE

2. SYSTEM CONFIGURATION BACKUP DIRECTORY
   - A backup folder is provided directly on your desktop containing original and 
     modified configuration scripts (for basebrd, visual layout assets, and system 
     icon .dll engines). 
   - Execute the corresponding `.cmd` scripts to instantly re-apply your mods if a 
     Windows Update block resets your configurations.

3. VISUAL THEME INTERACTION
   - For editions featuring premium desktop themes (like Morphs_Dragons_Light and 
     Morphs_Dragons_Dark), deploy styles directly using the quick desktop toggle links.
   - Legacy styling structures can be configured seamlessly using UltraUXThemePatcher, 
     executable inside 'Start Menu > Programs > Morph's Apps'.

4. LTSC APPS & MICROSOFT STORE PROVISIONING
   - For bare LTSC system editions lacking stock application suites, deploy the core 
     Microsoft Store infrastructure over an active internet connection.
   - Open PowerShell with Administrator permissions and execute: wsreset -i
   - Once complete, optimize your workflow by deploying the App Installer engine 
     (WinGet integration wrapper) directly at: https://microsoft.com
